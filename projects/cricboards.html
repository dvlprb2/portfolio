Project information

  • Category: Side Project
  • Source Code: GitHub
  • Date: 20 May, 2023
  • Status: Ongoing

This portfolio project utilizes React Native and Firebase to develop a mobile application that provides users with cricket match scoreboards, player profiles, and the ability to create clubs for playing cricket with friends. The application offers real-time updates of cricket match scores, allowing users to stay up-to-date with the latest results. Users can also explore player profiles to view statistics, performance history, and other relevant information. Additionally, the application enables users to create and manage cricket clubs, facilitating easy organization and coordination for friendly matches. By leveraging the power of React Native and Firebase, this mobile application delivers a seamless and engaging experience for cricket enthusiasts.
